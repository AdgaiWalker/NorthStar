# M9 关于我页面规范

## 页面概述
- **路由**: `/about`
- **优先级**: Phase 4（社区与编辑）
- **核心目的**: 展示创始人信息，引导用户加入社群
- **设计风格**: 简洁温馨，体现共建共享理念

## 功能模块

### 1. Header
- **返回按钮**
  - 图标：左箭头
  - 点击：返回上一页或首页
- **标题**
  - "关于我"

### 2. 个人介绍区域
- **布局**
  - 居中显示
  - 最大宽度：600px
- **内容**
  - **头像**
    - 尺寸：120x120px（移动端）/ 160x160px（桌面端）
    - 圆形
    - 阴影效果
    - 可选：悬浮时轻微放大
  - **姓名/昵称**
    - 字号：xl-2xl
    - 加粗
  - **个人简介**
    - 200-300字
    - 内容要求：
      - 体现共建共享理念
      - 说明创建"盘根"的初衷
      - 表达对用户的期待
    - 行高：1.75
    - 字号：16px
    - 居中对齐（或左对齐）
  - **个人标签（可选）**
    - 示例：产品经理、AI爱好者、终身学习者
    - 标签样式：圆角边框，浅色背景

### 3. 联系方式区域
- **标题**
  - "联系方式"
- **列表项**
  - 每项包含：
    - 图标（抖音/B站/邮箱/微博等）
    - 平台名称
    - 账号名称或链接
  - 点击：
    - 外部链接：弹出提示"即将跳转到外部网站"，确认后跳转
    - 复制账号：点击复制到剪贴板 + Toast提示
- **样式**
  - 卡片列表
  - 每项hover：背景变浅 + 轻微上移
  - 图标彩色（使用平台品牌色）

### 4. 社交媒体（可选）
- **内容**
  - 抖音
    - 图标：📱
    - 账号：@xxx
    - 点击：复制账号或跳转到抖音
  - B站
    - 图标：📺
    - 账号：@xxx
    - 点击：跳转到B站主页
  - 微博（可选）
  - 小红书（可选）
- **交互**
  - 点击账号：复制到剪贴板
  - 点击图标：跳转到对应平台

### 5. 加入社群区域
- **标题**
  - "加入社群"
  - 副标题："和志同道合的伙伴一起探索AI"
- **微信社群二维码**
  - 尺寸：200x200px
  - 边距：24px
  - 阴影效果
  - 图片描述：二维码下方显示文字"扫码加入盘根社群"
- **注意事项**
  - 提示文字："二维码7天有效，过期请联系管理员"
  - 小字号，灰色
- **备选方案（如二维码过期）**
  - 显示微信号
  - 点击复制微信号 + Toast提示"请添加微信：xxx"

### 6. 其他信息（可选）
- **网站统计（可选）**
  - 用户数量
  - 文章数量
  - 工具数量
  - 访问量
- **版本信息**
  - 当前版本：v5.3
  - 更新日志链接
- **友情链接**
  - 合作伙伴网站
  - 推荐资源

### 7. Footer（页脚）
- **内容**
  - ICP备案号：蜀ICP备2025165644号-1
  - 公安备案：备案中
  - 版权信息：© 2024 盘根 AI 指南针
  - 用户协议 | 隐私政策

## 状态管理

```typescript
interface AboutPageState {
  // 个人信息
  profile: {
    avatar: string;
    name: string;
    bio: string;
    tags: string[];
  };
  
  // 联系方式
  contacts: {
    platform: 'douyin' | 'bilibili' | 'weibo' | 'xiaohongshu' | 'email';
    icon: string;
    label: string;
    value: string; // 账号名或链接
    url?: string; // 外部链接
  }[];
  
  // 社群二维码
  qrCode: {
    imageUrl: string;
    expiresAt?: string; // 过期时间
  };
  
  // 微信号（备用）
  wechatId?: string;
  
  // 网站统计（可选）
  stats?: {
    users: number;
    articles: number;
    tools: number;
    views: number;
  };
  
  // UI状态
  showExternalLinkWarning: boolean;
  externalLink: string;
}
```

## API接口

### 获取个人信息
```
GET /api/about/profile
Response: {
  profile: Profile,
  contacts: Contact[],
  qrCode: QRCode,
  stats: Stats
}
```

### 统计访问（可选）
```
POST /api/about/visit
Response: { success: boolean }
```

## 响应式设计

### 移动端 (<768px)
- 单列布局
- 头像：120x120px
- 二维码：200x200px
- 联系方式垂直堆叠
- 内容居中对齐

### 平板 (768px-1024px)
- 内容区最大宽度：600px，居中
- 头像：140x140px

### 桌面 (>1024px)
- 内容区最大宽度：700px，居中
- 头像：160x160px
- 联系方式可改为网格布局（2列）

## 动画规范
- 页面加载：fade-in 500ms
- 头像加载：fade-in + scale 300ms
- 联系方式卡片：stagger fade-in（每个延迟50ms）
- Hover效果：transform translateY(-2px) 150ms

## 交互反馈
- 点击复制：Light Haptic + Toast:"已复制"
- 外部链接：弹窗提示
- 二维码加载失败：显示微信号备选方案

## 外部链接警告弹窗
- **触发条件**
  - 点击外部链接（抖音/B站等）
- **内容**
  - 标题："即将跳转到外部网站"
  - 描述："您将离开盘根 AI 指南针，跳转到 {platform}"
  - 按钮：
    - [取消]
    - [继续访问]（主按钮）
- **交互**
  - 点击[取消]：关闭弹窗
  - 点击[继续访问]：打开新标签页跳转

## 性能优化
- 头像图片压缩（WebP格式 + 备用JPG）
- 二维码懒加载
- 预加载关键图片

## 错误处理
- 二维码加载失败：显示微信号备选方案
- 外部链接失效：Toast提示"链接已失效"
- 网络错误：显示重试按钮

## SEO优化
- 页面标题："关于我 - 盘根 AI 指南针"
- Meta描述：包含创始人信息和网站理念
- Schema.org Person标记

## 埋点需求
- 页面访问量（PV/UV）
- 联系方式点击率（各平台统计）
- 二维码扫描次数（估算，通过页面停留时间）
- 外部链接跳转统计
- 复制操作统计
