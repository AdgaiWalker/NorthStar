## 技术栈

### 前端
- React 19 + TypeScript
- Vite
- TailwindCSS
- Headless UI
- Framer Motion（动画）
- Three.js + @react-three/fiber（3D动画）
- react-markdown（Markdown渲染）
- Markdown编辑器（待选型）
- react-i18next（国际化）

### 后端
- NestJS + TypeScript
- PostgreSQL（数据库）
- Redis（缓存）
- Prisma（ORM）

### AI服务
- 智谱AI (GLM-4.6) - 主要模型
- 支持多模型切换（Kimi/MiniMax/DeepSeek/Qwen）

### 部署
- 前端：Vercel / Netlify
- 后端：待定
- 数据库：云数据库

## 共通设计规范

### 主题系统
- 浅色模式
- 护眼模式（#F7F6F2）
- 深色模式（延后）

### 国际化
- 支持语言：中文（默认）、日语、英语、俄语
- 使用react-i18next

### 响应式断点
- 移动端：<768px
- 平板：768px-1024px
- 桌面：>1024px

### 动画规范
- 页面加载：fade-in 500ms
- 卡片hover：scale(1.02) + shadow 150ms
- 按钮点击：scale(0.95) 100ms
- 模态框：fade + scale 300ms

### 交互反馈（移动端）
- Light Haptic：轻量操作（点击、切换）
- Medium Haptic：中等操作（选择、添加）
- Success Haptic：成功反馈
- Warning Haptic：警告提示
- Error Haptic：错误提示

### 颜色系统
- 主色：蓝色 #2563EB
- 次要色：石板灰 #475569
- 护眼背景：#F7F6F2
- 护眼深色：#EBE9E0

### 圆角系统
- 小：4px
- 中：8px
- 大：12px
- 超大：16px-24px

### 阴影系统
- sm：轻阴影
- md：中等阴影
- lg：深阴影
- xl：超深阴影

## 开发建议

### 开发顺序（建议）
1. **Phase 1**：M8登录注册 + 用户系统
2. **Phase 2**：
   - M1首页（工具推荐）
   - M2搜索页（AI搜索）
   - M5生成方案（AI核心功能）
   - M3阅读页（Markdown渲染）
   - M6个人中心（用户数据管理）
3. **Phase 3**：M10支付页面（变现机制）
4. **Phase 4**：M14编辑后台 + M9关于我（社区运营）

### 复用组件
- Header（统一头部）
- Footer（统一页脚）
- Button（按钮组件）
- Input（输入框）
- Card（卡片容器）
- Modal（模态框）
- Toast（提示）
- Loading（加载状态）
- Empty（空状态）

### 状态管理
- 建议使用：Zustand 或 Redux Toolkit
- 全局状态：用户信息、主题、语言
- 页面状态：各页面独立管理

### API设计
- RESTful风格
- 统一错误处理
- 统一响应格式
- JWT认证
