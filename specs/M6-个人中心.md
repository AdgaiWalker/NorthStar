# M6/M7 个人中心规范

## 页面概述
- **路由**: `/user` 或 `/profile`
- **优先级**: Phase 2（核心搜索与方案生成）
- **依赖**: 用户认证系统
- **访问权限**: 仅登录用户

## 功能模块

### 1. Header
- **返回按钮**
  - 返回首页或上一页
- **标题**
  - "个人中心"
- **设置按钮**
  - 图标：齿轮
  - 点击：跳转到设置页面

### 2. 用户信息卡片
- **位置**
  - 页面顶部
- **内容**
  - 用户头像（可点击更换）
  - 用户名（可编辑）
  - 邮箱/手机号
  - 认证标识（普通用户/创作者/管理员）
  - 会员状态
    - 试用期剩余天数
    - 或付费会员到期时间
    - 或投稿贡献状态
- **操作**
  - 编辑资料按钮
  - 会员续费/升级按钮（试用期到期后）

### 3. Tab导航栏
- **选项**
  - 我的方案
  - 我的工具
  - 我的收藏
  - 我的创作（条件显示，仅创作者/管理员）
- **交互**
  - 点击切换Tab
  - 下划线指示当前Tab
  - 内容区平滑过渡（fade 200ms）
- **状态管理**
  - `currentTab: 'solutions' | 'tools' | 'favorites' | 'creations'`

### 4. 我的方案Tab
- **内容**
  - 方案列表（按创建时间倒序）
  - 每个方案卡片包含：
    - 方案标题
    - 创建时间
    - 涉及的工具（图标列表）
    - 步骤数量
    - 操作按钮
      - 查看详情
      - 编辑
      - 删除
      - 导出（Markdown/PDF/CSV）
- **交互**
  - 点击方案：跳转到方案详情页（M5）
  - 点击编辑：进入方案编辑模式
  - 点击删除：确认弹窗 → 删除方案
  - 点击导出：弹出导出格式选择
- **空状态**
  - 插图 + 文字："还没有保存的方案"
  - 引导按钮："去探索工具"
- **API需求**
  - `GET /api/user/solutions` - 获取用户方案列表
  - `DELETE /api/solutions/:id` - 删除方案
  - `GET /api/solutions/:id/export?format={md|pdf|csv}` - 导出方案

### 5. 我的工具Tab
- **内容**
  - 工具列表（按最后使用时间倒序）
  - 每个工具卡片包含：
    - 工具图标
    - 工具名称
    - 简短描述
    - 最后使用时间
    - 使用次数统计
    - 操作按钮
      - 查看详情
      - 移除
- **筛选/排序**
  - 排序：最近使用/使用频率/添加时间
  - 筛选：按领域
- **交互**
  - 点击工具：跳转到工具详情页
  - 点击移除：从我的工具中移除（确认提示）
- **空状态**
  - 插图 + 文字："还没有添加工具"
  - 引导按钮："去发现工具"
- **API需求**
  - `GET /api/user/tools` - 获取用户工具列表（含使用统计）
  - `DELETE /api/user/tools/:id` - 移除工具

### 6. 我的收藏Tab
- **内容**
  - 收藏的文章列表
  - 每个文章卡片包含：
    - 文章封面
    - 文章标题
    - 作者
    - 收藏时间
    - 操作按钮
      - 阅读
      - 取消收藏
- **筛选**
  - 按领域筛选
  - 按收藏时间排序
- **交互**
  - 点击文章：跳转到阅读页（M3）
  - 点击取消收藏：从列表移除
- **空状态**
  - 插图 + 文字："还没有收藏的文章"
  - 引导按钮："去阅读文章"
- **API需求**
  - `GET /api/user/favorites` - 获取收藏列表
  - `DELETE /api/user/favorites/:articleId` - 取消收藏

### 7. 我的创作Tab（条件显示）
- **显示条件**
  - 用户角色为`creator`或`admin`
- **内容**
  - 我的投稿文章列表
  - 每个文章卡片包含：
    - 文章标题
    - 状态标签（草稿/待审核/已发布/已拒绝）
    - 提交时间
    - 浏览量/点赞数（已发布的）
    - 操作按钮
      - 编辑
      - 删除
      - 查看统计（已发布）
- **筛选**
  - 状态筛选（全部/草稿/待审核/已发布）
- **交互**
  - 点击编辑：跳转到编辑器
  - 点击删除：确认后删除
  - 点击查看统计：弹出数据详情
- **入口**
  - "创作新文章"按钮（悬浮或顶部）
  - 点击：跳转到Markdown编辑器
- **空状态**
  - 插图 + 文字："开始创作你的第一篇文章"
  - 引导按钮："开始创作"
- **API需求**
  - `GET /api/user/articles` - 获取用户创作列表
  - `DELETE /api/articles/:id` - 删除文章
  - `GET /api/articles/:id/stats` - 获取文章统计

### 8. 设置入口
- **位置**
  - Tab导航栏最右侧
  - 或用户信息卡片中
- **点击**
  - 跳转到设置页面（独立页面）
- **设置页内容**
  - 账号设置
    - 修改头像
    - 修改用户名
    - 绑定/解绑邮箱/手机号
    - 修改密码
  - 偏好设置
    - 语言选择
    - 主题模式（浅色/护眼/深色）
    - 通知设置
  - 隐私设置
    - 方案可见性（公开/仅自己）
    - 收藏可见性
  - 关于
    - 应用版本
    - 用户协议
    - 隐私政策
    - 退出登录
  - 会员管理
    - 查看会员信息
    - 续费/升级
    - 退款申请（如适用）
  - 编辑后台入口（仅创作者/管理员）
    - 点击：跳转到M14编辑后台

## 状态管理

```typescript
interface UserCenterState {
  // 用户信息
  user: User | null;
  isLoadingUser: boolean;
  
  // Tab状态
  currentTab: 'solutions' | 'tools' | 'favorites' | 'creations';
  
  // 数据列表
  solutions: UserSolution[];
  tools: Tool[];
  favorites: Article[];
  creations: Article[];
  
  // 加载状态
  isLoadingSolutions: boolean;
  isLoadingTools: boolean;
  isLoadingFavorites: boolean;
  isLoadingCreations: boolean;
  
  // 编辑状态
  isEditingProfile: boolean;
}

interface User {
  id: string;
  username: string;
  email: string;
  phone?: string;
  avatar: string;
  role: 'user' | 'creator' | 'admin';
  membershipStatus: 'trial' | 'paid' | 'contributor' | 'expired';
  trialDaysLeft?: number;
  membershipExpiresAt?: string;
  contributionCount?: number; // 投稿数量（用于贡献者）
}
```

## API接口

### 获取用户信息
```
GET /api/user/profile
Response: User
```

### 更新用户信息
```
PUT /api/user/profile
Request: Partial<User>
Response: User
```

### 上传头像
```
POST /api/user/avatar
Request: FormData (file)
Response: { avatarUrl: string }
```

### 获取各Tab数据
```
GET /api/user/solutions
GET /api/user/tools
GET /api/user/favorites
GET /api/user/articles (我的创作)
```

### 删除操作
```
DELETE /api/solutions/:id
DELETE /api/user/tools/:id
DELETE /api/user/favorites/:articleId
DELETE /api/articles/:id
```

### 导出方案
```
GET /api/solutions/:id/export?format={md|pdf|csv}
Response: File download
```

## 响应式设计

### 移动端 (<768px)
- 单列布局
- Tab导航固定在顶部，可横向滚动
- 卡片列表垂直堆叠
- 操作按钮下拉菜单（省空间）

### 平板 (768px-1024px)
- 工具/收藏可改为网格布局（2列）
- Tab导航居中显示

### 桌面 (>1024px)
- 最大宽度：1200px
- 工具/收藏网格布局（3列）
- 方案卡片展示更多信息

## 动画规范
- Tab切换：fade 200ms
- 卡片加载：stagger fade-in（每个卡片延迟50ms）
- 删除操作：fade-out + slide-up 300ms
- 导出成功：按钮变为勾选图标，持续1秒

## 交互反馈
- Tab切换：Light Haptic
- 删除操作：Warning Haptic + 确认弹窗
- 导出成功：Success Haptic + Toast
- 取消收藏：Light Haptic

## 性能优化
- 列表分页加载（每页20条）
- 图片懒加载
- Tab内容按需加载（切换时才请求数据）
- 缓存已加载的Tab数据（5分钟）

## 错误处理
- 加载失败：显示重试按钮
- 删除失败：Toast提示 + 回滚操作
- 导出失败：Toast提示 + 允许重试

## 埋点需求
- Tab切换频率统计
- 方案查看/编辑/删除/导出行为
- 工具使用频率分析
- 收藏内容类型分析
- 创作文章状态分布
- 设置项修改频率
