# M2/M4 搜索页规范

## 页面概述
- **路由**: `/search?q={query}`
- **优先级**: Phase 2（核心搜索与方案生成）
- **依赖**: AI服务接口、工具数据库
- **设计参考**: 智能推荐 + 手动选择结合

## 功能模块

### 1. Header（搜索页头部）
- **返回按钮**
  - 图标：左箭头
  - 点击：返回上一页或首页
- **标题**
  - 显示"搜索结果"
  - 移动端可隐藏，仅保留返回按钮
- **清空按钮**
  - 文字："清空"或垃圾桶图标
  - 点击：清空所有已选工具

### 2. AI推荐区域
- **显示条件**
  - 当用户输入问题（非空）且AI返回推荐时显示
  - 使用卡片样式，与工具列表区分
- **内容**
  - 标题：🤖 AI 推荐
  - AI理解摘要："根据你的问题，建议使用："
  - 推荐工具列表（最多3个）
    - 工具名称
    - 推荐理由（简短）
  - 操作按钮
    - "查看详细方案"：跳转到M5生成方案页（自动填充推荐工具）
    - "全部添加"：一键添加所有推荐工具到已选列表
- **API需求**
  - `POST /api/ai/recommend`
    - Request: `{ query: string, tools: Tool[] }`
    - Response: `{ summary: string, recommendation: string, suggestedTools: string[] }`
- **加载状态**
  - 骨架屏或旋转加载器
  - 文字："AI正在分析..."
- **错误处理**
  - AI服务不可用：降级为关键词搜索
  - 显示提示："AI服务暂时不可用，已切换到关键词搜索"

### 3. 搜索模式切换
- **选项**
  - AI模式（默认）
  - 关键词模式
- **交互**
  - Tab切换样式
  - 切换时重新执行搜索
- **状态管理**
  - `searchMode: 'ai' | 'keyword'`

### 4. 工具列表区域
- **标题**
  - "所有工具" 或 "搜索结果"
  - 显示结果数量：如"找到 12 个工具"
- **排序/筛选**
  - 排序选项：
    - 相关度（默认）
    - 评分
    - 使用人数
  - 筛选选项：
    - 领域（creative/dev/work）
    - 标签（多选）
    - 评分范围
- **工具卡片**
  - 布局：列表视图（移动端）/ 网格视图（桌面端可切换）
  - 内容：
    - 工具图标/封面
    - 工具名称
    - 简短描述
    - 评分 + 使用人数
    - 标签（最多3个）
    - 选择按钮
      - 未选择：[添加] 按钮（边框样式）
      - 已选择：[已选择] 按钮（填充样式 + 勾选图标）
- **交互**
  - 点击卡片：跳转到工具详情页
  - 点击[添加]按钮：
    - 按钮变为[已选择]
    - 底部已选数量 +1
    - 200ms fade + scale 动画
    - Light Haptic 反馈
  - 点击[已选择]按钮：
    - 按钮恢复[添加]
    - 底部已选数量 -1
    - 200ms fade 动画
    - Light Haptic 反馈
- **空状态**
  - 无搜索结果：
    - 插图 + 提示文字
    - "未找到相关工具，试试换个关键词？"
    - 推荐热门工具（3-5个）

### 5. 底部操作栏（固定）
- **位置**
  - 固定在页面底部
  - 毛玻璃背景 + 阴影
- **内容**
  - 左侧：已选工具数量
    - "已选 3 个工具"
    - 点击可展开查看已选工具列表
  - 右侧：生成方案按钮
    - 主按钮样式（渐变背景）
    - 文字："生成方案 →"
    - 禁用条件：已选工具数量 = 0
- **交互**
  - 点击"生成方案"：
    - 跳转到M5生成方案页
    - 传递已选工具ID列表
  - 点击已选数量：
    - 从底部弹出已选工具抽屉
    - 显示已选工具列表 + 快速移除功能

### 6. 已选工具抽屉（可选）
- **触发**
  - 点击底部操作栏的"已选 X 个工具"
- **内容**
  - 标题："已选工具"
  - 工具列表（横向滚动）
    - 工具图标 + 名称
    - 移除按钮（X图标）
  - 操作按钮
    - "清空所有"
    - "生成方案"
- **交互**
  - 点击工具：跳转到工具详情页
  - 点击移除：从已选列表移除
  - 点击外部区域：关闭抽屉

## 状态管理

```typescript
interface SearchPageState {
  // 搜索相关
  query: string;
  searchMode: 'ai' | 'keyword';
  
  // 工具列表
  searchResults: Tool[];
  isLoading: boolean;
  
  // AI推荐
  aiRecommendation: {
    summary: string;
    recommendation: string;
    suggestedTools: string[];
  } | null;
  aiLoading: boolean;
  
  // 已选工具
  selectedToolIds: Set<string>;
  
  // 筛选排序
  sortBy: 'relevance' | 'rating' | 'usage';
  filterDomain: Domain | 'all';
  filterTags: string[];
  
  // UI状态
  showSelectedDrawer: boolean;
  viewMode: 'list' | 'grid'; // 桌面端
}
```

## API接口

### AI智能推荐
```
POST /api/ai/recommend
Request: {
  query: string,
  availableTools: Tool[],
  availableArticles: Article[]
}
Response: {
  summary: string,
  recommendation: string,
  suggestedTools: string[], // 工具名称数组
  suggestedArticles: string[] // 文章标题数组
}
```

### 关键词搜索
```
GET /api/tools/search?q={query}&domain={domain}&tags={tags}&sort={sort}
Response: {
  total: number,
  tools: Tool[]
}
```

### 获取工具详情
```
GET /api/tools/{id}
Response: Tool
```

## 响应式设计

### 移动端 (<768px)
- 列表视图（单列）
- AI推荐卡片占满宽度
- 底部操作栏固定
- 筛选器收起在按钮中

### 平板 (768px-1024px)
- 网格视图（2列）
- AI推荐卡片居中显示

### 桌面 (>1024px)
- 网格视图（3-4列）
- 侧边筛选栏（可选）
- 最大宽度：1280px

## 动画规范
- AI推荐卡片加载：fade-in + slide-down 300ms
- 工具选择：scale(0.95) → scale(1) 200ms
- 底部操作栏：slide-up 300ms
- 抽屉弹出：slide-up + fade-in 250ms

## 交互反馈
- 工具添加/移除：Light Haptic
- 生成方案按钮点击：Medium Haptic
- AI推荐加载完成：Success Haptic

## 性能优化
- 搜索防抖：300ms
- 工具列表虚拟滚动（>50个结果）
- AI推荐缓存：相同query 5分钟内复用结果
- 图片懒加载

## 错误处理
- AI服务失败：自动降级到关键词搜索
- 网络错误：显示重试按钮
- 搜索超时：提示用户并允许重试

## 埋点需求
- 搜索使用量（AI模式 vs 关键词模式）
- AI推荐接受率（点击"查看详细方案"比例）
- 工具添加/移除行为
- 生成方案转化率
- 搜索结果为空的query（用于优化工具库）
