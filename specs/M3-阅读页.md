# M3 阅读页规范

## 页面概述
- **路由**: `/article/:id` 或 `/topic/:topicId/article/:articleId`
- **优先级**: Phase 2（核心搜索与方案生成）
- **依赖**: Markdown渲染、文章数据库
- **设计参考**: Obsidian（编辑模式）+ 飞书（移动端交互）

## 功能模块

### 1. 目录侧边栏（桌面端）
- **显示条件**
  - 仅桌面端显示（>1024px）
  - 当文章属于某个主题（Topic）时显示
- **位置**
  - 固定在左侧
  - 宽度：280px（阅读模式）/ 320px（编辑模式）
  - 高度：`calc(100vh - 64px)`
  - 可滚动
- **内容**
  - 返回按钮：返回首页
  - 主题信息
    - 主题标题
    - 主题描述（简短）
  - 文章列表
    - 序号 + 文章标题
    - 当前文章高亮（蓝色背景）
    - 其他文章：灰色文字，hover时背景变浅
- **交互**
  - 点击文章：切换到该文章（平滑过渡）
  - 滚动同步：当前阅读到的标题高亮

### 2. 移动端折叠目录
- **显示条件**
  - 移动端/平板端（<1024px）
  - 当文章属于某个主题时显示
- **触发**
  - 顶部"目录"按钮
  - 左滑手势
- **样式**
  - 从左侧滑出
  - 毛玻璃背景
  - 占满屏幕高度
- **交互**
  - 点击外部区域：关闭
  - 右滑手势：关闭
  - 点击文章：跳转并关闭目录

### 3. 面包屑导航
- **位置**
  - 文章顶部
  - 移动端可隐藏或简化
- **内容**
  - 首页 > 领域 > 主题（可选）> 当前文章
  - 每个部分可点击
- **交互**
  - 点击：跳转到对应页面

### 4. 文章Header
- **内容**
  - 文章标题（H1）
  - 元信息
    - 作者头像 + 名称
    - 发布日期
    - 阅读时间
    - 认证标识（如有）
- **样式**
  - 标题：3xl-4xl字号，加粗
  - 元信息：小字号，灰色

### 5. 视频播放器（条件渲染）
- **显示条件**
  - `article.isVideo === true`
- **内容**
  - 视频封面图
  - 播放按钮（中央大图标）
  - 时长标识（右下角）
- **交互**
  - 点击：播放视频（模拟或实际视频）
  - Hover：播放按钮放大（scale 1.1）
- **样式**
  - 宽高比：16:9
  - 圆角：12px
  - 阴影：lg

### 6. Markdown内容区域
- **渲染**
  - 使用`react-markdown`或类似库
  - 支持语法：
    - 标题（H1-H6）
    - 列表（有序/无序）
    - 代码块（语法高亮）
    - 表格
    - 引用
    - 图片
    - 链接
    - 粗体/斜体
- **样式**
  - 护眼模式：`prose-stone`类
  - 浅色模式：`prose-slate`类
  - 最大宽度：720px（桌面端居中）
  - 行高：1.75
  - 字号：16px-18px
- **代码高亮**
  - 使用`prism.js`或`highlight.js`
  - 支持多种语言
  - 行号显示（可选）

### 7. Markdown编辑器（编辑模式）
- **显示条件**
  - 用户角色为`creator`或`admin`
  - 点击"编辑"按钮后显示
- **布局**
  - 三列布局（桌面端）
    - 左：目录（280px）
    - 中：编辑器（flex 1，最大600px）
    - 右：实时预览（flex 1，最大720px）
- **编辑器功能**
  - 实时预览（左右分屏）
  - 语法高亮
  - 工具栏
    - 加粗（Ctrl+B）
    - 斜体（Ctrl+I）
    - 链接（Ctrl+K）
    - 图片插入
    - 代码块
    - 标题级别
  - 自动保存：每2秒自动保存草稿
  - 滚动同步：编辑区 ↔ 预览区
  - 字数统计（底部状态栏）
- **快捷键**
  - Ctrl/Cmd + B：加粗
  - Ctrl/Cmd + I：斜体
  - Ctrl/Cmd + K：插入链接
  - Ctrl/Cmd + S：手动保存
- **API需求**
  - `POST /api/articles/:id/draft` - 保存草稿
  - `PUT /api/articles/:id` - 发布更新

### 8. 大纲导航（编辑模式）
- **位置**
  - 右侧固定栏
  - 或折叠按钮（移动端）
- **内容**
  - 自动提取文章中的H1-H6标题
  - 树形结构显示
  - 当前阅读位置高亮
- **交互**
  - 点击标题：平滑滚动到对应位置

### 9. 阅读进度条
- **位置**
  - 页面顶部
  - 细线（2px高度）
- **颜色**
  - 蓝色渐变
- **交互**
  - 随滚动自动更新
  - 保存阅读位置（localStorage + 云端同步）

### 10. 底部工具栏
- **显示条件**
  - 移动端：点击展开（默认隐藏）
  - 桌面端：悬浮在右下角
- **内容**
  - 收藏按钮
    - 未收藏：空心爱心
    - 已收藏：填充爱心 + 红色
  - 分享按钮
    - 复制链接
    - 分享到微信
    - 分享到QQ
  - 工具列表按钮
    - 显示文章相关的工具
    - 点击可快速添加到"我的工具"
- **交互**
  - 点击收藏：
    - 切换收藏状态
    - Light Haptic
    - Toast提示："已收藏"或"已取消收藏"
  - 点击分享：弹出分享菜单
  - 点击工具列表：弹出工具抽屉

### 11. 评论区
- **位置**
  - 文章底部
  - 顶部有分隔线
- **内容**
  - 标题："评论 (X)"
  - 评论列表
    - 用户头像
    - 用户名
    - 评论内容
    - 发布时间
    - 点赞按钮
  - 发表评论输入框（登录后显示）
- **交互**
  - 点击发表：
    - 调用API发布评论
    - 刷新评论列表
    - 滚动到新评论位置
  - 点击点赞：
    - 数量+1，按钮变色
    - Light Haptic
- **API需求**
  - `GET /api/articles/:id/comments` - 获取评论列表
  - `POST /api/articles/:id/comments` - 发表评论
  - `POST /api/comments/:id/like` - 点赞评论

### 12. 相关推荐（可选）
- **位置**
  - 评论区下方
- **内容**
  - 标题："相关推荐"
  - 文章卡片列表（横向滚动）
    - 封面图
    - 标题
    - 阅读时间
- **逻辑**
  - 同主题的其他文章
  - 或同领域的热门文章
- **API需求**
  - `GET /api/articles/:id/related` - 获取相关文章

## 状态管理

```typescript
interface ArticlePageState {
  // 文章数据
  article: Article | null;
  isLoading: boolean;
  
  // 主题相关
  topic: Topic | null;
  topicArticles: Article[];
  
  // 编辑模式
  isEditMode: boolean;
  draftContent: string;
  autoSaveStatus: 'saved' | 'saving' | 'error';
  
  // UI状态
  showMobileToc: boolean; // 移动端目录
  showToolDrawer: boolean; // 工具列表抽屉
  readingProgress: number; // 阅读进度 0-100
  
  // 交互状态
  isFavorited: boolean;
  comments: Comment[];
  relatedArticles: Article[];
}
```

## API接口

### 获取文章详情
```
GET /api/articles/:id
Response: Article
```

### 获取主题信息及文章列表
```
GET /api/topics/:topicId/articles
Response: {
  topic: Topic,
  articles: Article[]
}
```

### 保存草稿
```
POST /api/articles/:id/draft
Request: { content: string }
Response: { status: 'success' }
```

### 发布文章
```
PUT /api/articles/:id
Request: { content: string, title: string }
Response: Article
```

### 收藏/取消收藏
```
POST /api/articles/:id/favorite
Response: { isFavorited: boolean }
```

### 评论相关
```
GET /api/articles/:id/comments
POST /api/articles/:id/comments
POST /api/comments/:id/like
```

## 响应式设计

### 移动端 (<768px)
- 单列布局，沉浸式阅读
- 目录折叠（左滑展开）
- 双指缩放调整字体
- 底部工具栏点击展开
- 编辑模式改为上下布局（编辑器在上，预览在下）

### 平板 (768px-1024px)
- 双列布局
- 左：目录（280px）
- 右：正文 + 浮动工具栏
- 支持手势操作

### 桌面 (>1024px)
- 阅读模式：双列（目录 + 正文）
- 编辑模式：三列（目录 + 编辑器 + 预览）
- 正文最大宽度：720px
- 工具栏悬浮右下角

## 动画规范
- 页面加载：fade-in 500ms
- 目录切换文章：fade 200ms + 内容平滑滚动
- 评论发布：slide-in-from-bottom 300ms
- 工具栏展开：slide-up 250ms

## 手势操作（移动端）
- 左滑：显示目录
- 右滑：返回上一页
- 双指缩放：调整字体大小
- 双击：快速回到顶部（实验性）

## 交互反馈
- 收藏成功：Success Haptic + Toast
- 评论发布：Medium Haptic
- 滚动到标题：平滑滚动动画

## 性能优化
- Markdown渲染缓存
- 图片懒加载
- 评论分页加载（首次加载10条）
- 代码高亮按需加载
- 阅读位置自动保存（防抖3秒）

## 错误处理
- 文章不存在：404页面
- 加载失败：显示重试按钮
- 评论发布失败：Toast提示 + 保留输入内容

## 埋点需求
- 阅读时长统计
- 滚动深度（25%、50%、75%、100%）
- 收藏/分享行为
- 评论互动率
- 相关推荐点击率
- 工具添加转化率
