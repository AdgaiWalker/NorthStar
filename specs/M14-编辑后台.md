# M14 编辑后台规范

## 页面概述
- **路由**: `/admin`
- **优先级**: Phase 4（社区与编辑）
- **依赖**: 用户权限系统
- **访问权限**: 仅`creator`和`admin`角色
- **设计参考**: 微信公众号后台

## 功能模块

### 1. 左侧导航栏
- **Logo区域**
  - 显示"盘根后台"
  - 点击返回首页
- **导航模块**
  - 内容管理（默认选中）
  - 用户管理（仅admin可见）
  - 数据统计
  - 系统设置
- **底部**
  - 返回前台按钮
  - 退出登录按钮
- **样式**
  - 宽度：240px
  - 可收缩（点击按钮切换）
  - 收缩后宽度：64px（仅显示图标）
- **交互**
  - 点击模块：切换右侧内容区
  - 当前模块高亮显示

### 2. 内容管理模块
#### 2.1 筛选栏
- **位置**
  - 内容区顶部
- **内容**
  - 状态筛选下拉框
    - 全部
    - 待审核
    - 已推荐
    - 已拒绝
    - 草稿箱
  - 搜索框
    - 占位符："搜索标题、作者..."
    - 实时搜索或回车搜索
  - 时间范围选择器（可选）
  - 领域筛选（creative/dev/work）

#### 2.2 待审核区域
- **标题**
  - "待审核 (X)"，显示待审核数量
- **文章列表**
  - 每个文章卡片包含：
    - 文章图标（📄或封面缩略图）
    - 文章标题（可点击查看详情）
    - 摘要（2行截断）
    - 元信息
      - 作者名
      - 提交日期
      - 领域标签
    - 操作按钮
      - [推荐]：推荐到精选区
      - [忽略]：标记为已查看但不推荐
      - [编辑]：修改文章内容
      - [详情]：查看完整内容
- **交互**
  - 点击[推荐]：
    - 弹出确认框："确定要推荐这篇文章吗？"
    - 确认后：
      - 200ms fade out动画
      - Medium Haptic
      - 移至"已推荐"区域
      - Toast提示："已推荐"
  - 点击[忽略]：
    - 直接移除（或移至"已忽略"列表）
    - Toast提示："已忽略"
  - 点击[编辑]：
    - 弹出编辑器模态框
    - 或跳转到编辑页面
  - 批量操作
    - 复选框多选
    - 底部浮动操作栏：批量推荐/忽略/删除

#### 2.3 已推荐区域
- **标题**
  - "已推荐 (X)"
- **文章列表**
  - 类似待审核列表
  - 操作按钮
    - [取消推荐]：移回待审核
    - [编辑]
    - [删除]：彻底删除（二次确认）
- **排序**
  - 可拖拽调整推荐顺序（影响前台显示顺序）
  - 推荐时间排序

#### 2.4 已拒绝区域
- **标题**
  - "已拒绝 (X)"
- **文章列表**
  - 包含拒绝原因
  - 操作按钮
    - [重新审核]：移回待审核
    - [删除]

#### 2.5 草稿箱
- **标题**
  - "草稿箱 (X)"
- **内容**
  - 管理员/创作者自己的草稿
  - 操作按钮
    - [编辑]
    - [发布]
    - [删除]

### 3. 用户管理模块（仅admin）
#### 3.1 用户列表
- **筛选**
  - 角色筛选（全部/普通用户/创作者/管理员）
  - 会员状态筛选（试用中/付费会员/过期）
  - 搜索用户名/邮箱
- **用户卡片**
  - 用户头像
  - 用户名
  - 邮箱
  - 角色标签
  - 会员状态
  - 注册时间
  - 最后登录时间
  - 操作按钮
    - [查看详情]
    - [编辑角色]
    - [禁用/启用]
    - [删除]

#### 3.2 权限设置
- **角色管理**
  - 为用户分配角色（user/creator/admin）
  - 自定义权限（可选）
- **黑名单**
  - 被禁用的用户列表
  - 可恢复

#### 3.3 投稿贡献者管理
- **列表**
  - 投稿贡献者列表
  - 显示贡献文章数量
- **操作**
  - 批准/拒绝贡献者申请
  - 查看贡献历史

### 4. 数据统计模块
#### 4.1 总览仪表盘
- **关键指标卡片**
  - 总用户数
  - 今日新增用户
  - 总文章数
  - 待审核文章数
  - 总浏览量
  - 总点赞数
- **图表**
  - 用户增长趋势（折线图）
  - 文章发布趋势（柱状图）
  - 领域分布（饼图）
  - 热门文章Top 10（列表）

#### 4.2 文章统计
- **每篇文章的详细数据**
  - 浏览量
  - 点赞数
  - 收藏数
  - 评论数
  - 平均阅读时长
- **排序**
  - 按浏览量/点赞数/收藏数排序

#### 4.3 用户行为统计
- **搜索热词**
  - 显示最常搜索的关键词
- **工具热度**
  - 最常被选择的工具
- **方案生成统计**
  - 方案生成次数
  - 工具组合分析

### 5. 系统设置模块
#### 5.1 AI模型配置
- **模型选择**
  - 下拉选择当前使用的AI模型
    - 智谱AI (GLM-4.6)
    - Kimi
    - MiniMax
    - DeepSeek
    - 千问
- **API密钥管理**
  - 输入框（masked）
  - 测试连接按钮
  - 保存按钮
- **模型参数**
  - Temperature
  - Max Tokens
  - 其他参数（按需）

#### 5.2 网站配置
- **基础信息**
  - 网站名称
  - 网站描述
  - Logo上传
  - 备案号
- **SEO设置**
  - 默认标题
  - 默认描述
  - 关键词

#### 5.3 支付配置（可选）
- **微信支付**
  - 商户ID
  - API密钥
- **支付宝**
  - 应用ID
  - 私钥

#### 5.4 通知设置
- **邮件通知**
  - SMTP配置
  - 通知模板
- **系统通知**
  - 新文章提交时通知管理员
  - 用户注册时通知

### 6. 文章详情模态框/页面
- **触发**
  - 点击文章标题或[详情]按钮
- **内容**
  - 完整文章内容（Markdown渲染）
  - 元信息
  - 作者信息
  - 提交时间
  - 审核历史（如有）
- **操作**
  - 推荐/取消推荐
  - 编辑
  - 拒绝（需填写原因）
  - 删除

### 7. 编辑器模态框/页面
- **内容**
  - Markdown编辑器（参考M3）
  - 标题输入框
  - 领域选择
  - 标签输入
  - 封面上传
- **操作**
  - 保存草稿
  - 发布
  - 取消

## 状态管理

```typescript
interface AdminState {
  // 导航
  currentModule: 'content' | 'users' | 'stats' | 'settings';
  sidebarCollapsed: boolean;
  
  // 内容管理
  contentFilter: {
    status: 'all' | 'pending' | 'recommended' | 'rejected' | 'draft';
    domain: Domain | 'all';
    searchQuery: string;
    dateRange?: { start: string; end: string };
  };
  pendingArticles: Article[];
  recommendedArticles: Article[];
  rejectedArticles: Article[];
  draftArticles: Article[];
  selectedArticleIds: Set<string>; // 批量操作
  
  // 用户管理
  userFilter: {
    role: 'all' | 'user' | 'creator' | 'admin';
    membershipStatus: 'all' | 'trial' | 'paid' | 'expired';
    searchQuery: string;
  };
  users: User[];
  
  // 数据统计
  statsData: {
    overview: OverviewStats;
    articleStats: ArticleStats[];
    userBehavior: UserBehaviorStats;
  };
  
  // 系统设置
  systemConfig: SystemConfig;
  
  // 加载状态
  isLoading: boolean;
}

interface Article {
  id: string;
  title: string;
  summary: string;
  content: string;
  author: string;
  authorId: string;
  domain: Domain;
  tags: string[];
  coverUrl?: string;
  status: 'pending' | 'recommended' | 'rejected' | 'draft';
  submittedAt: string;
  reviewedAt?: string;
  rejectionReason?: string;
  stats?: {
    views: number;
    likes: number;
    comments: number;
  };
}
```

## API接口

### 内容管理
```
GET /api/admin/articles?status={status}&domain={domain}&q={query}
PUT /api/admin/articles/:id/recommend
PUT /api/admin/articles/:id/reject { reason: string }
PUT /api/admin/articles/:id/ignore
DELETE /api/admin/articles/:id
PUT /api/admin/articles/:id { content, title, domain, tags }
POST /api/admin/articles/batch { action: 'recommend'|'reject'|'delete', articleIds: string[] }
```

### 用户管理
```
GET /api/admin/users?role={role}&status={status}&q={query}
PUT /api/admin/users/:id/role { role: 'user'|'creator'|'admin' }
PUT /api/admin/users/:id/ban
PUT /api/admin/users/:id/unban
DELETE /api/admin/users/:id
```

### 数据统计
```
GET /api/admin/stats/overview
GET /api/admin/stats/articles
GET /api/admin/stats/user-behavior
```

### 系统设置
```
GET /api/admin/settings
PUT /api/admin/settings { ...config }
POST /api/admin/settings/ai-model/test { model: string, apiKey: string }
```

## 响应式设计

### 移动端 (<768px)
- 左侧导航改为顶部Tab栏
- 内容列表垂直堆叠
- 筛选栏改为抽屉式
- 数据统计卡片单列显示

### 平板 (768px-1024px)
- 侧边栏可收缩
- 内容列表可显示更多信息

### 桌面 (>1024px)
- 标准左右布局
- 内容区最大宽度：1400px
- 列表可显示完整信息

## 动画规范
- 侧边栏展开/收缩：width过渡 300ms
- 文章推荐：fade-out 200ms + 移至已推荐区
- 批量操作：loading spinner
- 骨架屏加载：shimmer动画

## 交互反馈
- 推荐操作：Medium Haptic + Toast
- 删除操作：Warning Haptic + 确认弹窗
- 保存设置：Success Haptic + Toast

## 权限控制
- **Creator角色**
  - 仅可查看和管理自己提交的文章
  - 无用户管理权限
  - 无系统设置权限
- **Admin角色**
  - 完全权限

## 性能优化
- 文章列表分页加载（每页20条）
- 骨架屏加载状态
- 图片懒加载
- 数据缓存（5分钟）

## 错误处理
- 操作失败：Toast提示 + 回滚状态
- 网络错误：显示重试按钮
- 权限不足：跳转到403页面

## 埋点需求
- 审核操作统计（推荐/拒绝/忽略频率）
- 批量操作使用率
- 各模块访问频率
- AI模型切换记录
- 设置修改记录
