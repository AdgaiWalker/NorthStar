# M8 登录/注册页规范

## 页面概述
- **路由**: `/login` 或 `/auth`
- **优先级**: Phase 1（用户系统与基础功能）
- **核心特色**: 3D创意设计，体现"盘根"共建共享理念
- **技术要求**: WebGL/Three.js（桌面端）

## 功能模块

### 1. 3D动画区域（桌面端）
- **技术实现**
  - Three.js + @react-three/fiber
  - WebGL渲染
  - 移动端降级为2D动画或静态图
- **视觉设计**
  - 主题：根系网络从中心向外扩散
  - 节点：表示用户，动态连接
  - 颜色：渐变色（蓝色到紫色）
  - 动画：
    - 根系生长动画（页面加载时播放一次）
    - 节点之间连接线动态形成
    - 粒子效果（可选）
- **交互**
  - 鼠标移动：产生连接线跟随效果
  - Hover节点：节点放大 + 连接线高亮
  - 点击节点：触发彩蛋动画（可选）
- **性能优化**
  - 限制粒子数量（<500）
  - 使用LOD（Level of Detail）
  - 移动端自动禁用，显示2D备选方案
- **寓意**
  - 每个用户都是网络的一个节点
  - 连接代表知识共享和协作

### 2. 表单区域
#### 2.1 欢迎文案
- **标题**
  - "欢迎来到盘根"（字号：2xl-3xl）
- **副标题**
  - "共建共享，连接 AI 世界"
  - 灰色小字，体现理念

#### 2.2 Tab切换
- **选项**
  - 登录
  - 注册
- **交互**
  - 点击切换表单内容
  - 下划线指示当前Tab
  - 切换动画：fade 200ms

#### 2.3 登录表单
- **输入框1：邮箱/手机号**
  - 图标：📧
  - 占位符："邮箱或手机号"
  - 验证：
    - 邮箱格式校验（正则）
    - 手机号格式校验（11位数字）
  - 错误提示："请输入有效的邮箱或手机号"
- **输入框2：密码**
  - 图标：🔐
  - 占位符："密码"
  - 类型：password（可切换显示/隐藏）
  - 验证：非空
  - 错误提示："请输入密码"
- **记住我复选框**
  - 勾选后：保存登录状态（localStorage + JWT）
- **忘记密码链接**
  - 点击：跳转到密码重置页面
- **登录按钮**
  - 主按钮样式（渐变背景）
  - 文字："登录"
  - 点击：
    - 调用登录API
    - 显示loading状态（按钮内旋转图标）
    - 成功：跳转到首页或上一页
    - 失败：Toast提示错误信息

#### 2.4 注册表单
- **输入框1：用户名**
  - 图标：👤
  - 占位符："用户名（2-20字符）"
  - 验证：
    - 长度：2-20字符
    - 唯一性检查（实时或提交时）
  - 错误提示："用户名已存在"或"用户名长度不符"
- **输入框2：邮箱/手机号**
  - 同登录表单
  - 额外验证：唯一性检查
  - 错误提示："该邮箱/手机号已注册"
- **输入框3：密码**
  - 图标：🔐
  - 占位符："密码（至少6位）"
  - 验证：
    - 长度≥6
    - 强度提示（可选）：弱/中/强
  - 实时反馈：密码强度条
- **输入框4：确认密码**
  - 占位符："再次输入密码"
  - 验证：与密码一致
  - 错误提示："两次密码不一致"
- **验证码输入框**
  - 占位符："验证码"
  - 发送验证码按钮：
    - 点击发送验证码到邮箱/手机
    - 60秒倒计时："60s后重新发送"
    - 倒计时结束后可重新发送
- **用户协议复选框**
  - 文字："我已阅读并同意《用户协议》和《隐私政策》"
  - 链接可点击，弹窗显示协议内容
  - 必须勾选才能注册
- **注册按钮**
  - 主按钮样式
  - 文字："注册"
  - 点击：
    - 验证所有字段
    - 调用注册API
    - 成功：自动登录 + 跳转到首页
    - 失败：Toast提示错误信息

#### 2.5 第三方登录
- **分隔线**
  - 文字："或使用第三方登录"
- **第三方按钮**
  - 微信登录
    - 图标：微信图标
    - 点击：拉起微信授权（二维码或直接跳转）
  - QQ登录
    - 图标：QQ图标
    - 点击：拉起QQ授权
- **样式**
  - 图标按钮，圆形或方形
  - Hover效果：放大 + 阴影
- **流程**
  - 点击 → 跳转第三方授权页
  - 授权成功 → 回调到`/auth/callback?code=xxx`
  - 后端换取token
  - 前端保存token + 跳转首页

### 3. 密码重置页面（独立页面）
- **路由**: `/reset-password`
- **流程**
  1. 输入邮箱/手机号 → 发送验证码
  2. 输入验证码 → 验证通过
  3. 输入新密码 + 确认密码
  4. 提交 → 重置成功 → 跳转登录页

### 4. 2D备选方案（移动端）
- **显示条件**
  - 屏幕宽度 <768px
  - 或设备不支持WebGL
- **内容**
  - 静态背景图（根系网络插画）
  - 或简单的CSS动画（节点淡入淡出）
- **样式**
  - 毛玻璃效果叠加
  - 保持视觉一致性

## API接口

### 登录
```
POST /api/auth/login
Request: {
  account: string, // 邮箱或手机号
  password: string,
  rememberMe: boolean
}
Response: {
  token: string,
  user: User
}
```

### 注册
```
POST /api/auth/register
Request: {
  username: string,
  account: string,
  password: string,
  verifyCode: string
}
Response: {
  token: string,
  user: User
}
```

### 发送验证码
```
POST /api/auth/send-code
Request: {
  account: string, // 邮箱或手机号
  type: 'register' | 'reset-password'
}
Response: {
  success: boolean,
  expiresIn: number // 有效期（秒）
}
```

### 用户名/账号唯一性检查
```
GET /api/auth/check-unique?field={username|email|phone}&value={value}
Response: {
  available: boolean
}
```

### 第三方登录
```
GET /api/auth/oauth/{provider}?redirect_uri={uri}
// 跳转到第三方授权页

GET /api/auth/oauth/callback?code={code}&state={state}
// 回调处理，返回token
```

### 密码重置
```
POST /api/auth/reset-password
Request: {
  account: string,
  verifyCode: string,
  newPassword: string
}
Response: {
  success: boolean
}
```

## 响应式设计

### 移动端 (<768px)
- 禁用3D动画，使用2D背景
- 表单区域占满屏幕
- 输入框全宽显示
- 按钮全宽显示

### 平板 (768px-1024px)
- 3D动画简化（降低粒子数）
- 表单区域居中，最大宽度：500px

### 桌面 (>1024px)
- 左右分栏布局
  - 左：3D动画（60%）
  - 右：表单（40%）
- 表单区域最大宽度：450px
- 完整3D效果

## 动画规范
- 页面加载：3D动画播放根系生长 2s
- Tab切换：fade 200ms
- 表单提交：按钮loading旋转
- 错误提示：shake 300ms
- 登录成功：淡出 + 跳转

## 交互反馈
- 输入框聚焦：边框变色 + 阴影
- 按钮点击：Light Haptic
- 登录成功：Success Haptic + Toast
- 验证失败：Error Haptic + Toast
- 验证码发送：Toast提示"验证码已发送"

## 表单验证规则
- **邮箱**: 正则 `/^[^\s@]+@[^\s@]+\.[^\s@]+$/`
- **手机号**: 正则 `/^1[3-9]\d{9}$/` (中国大陆)
- **用户名**: 2-20字符，字母/数字/中文/下划线
- **密码**: 至少6位
- **验证码**: 6位数字

## 安全措施
- HTTPS传输
- 密码加密（前端可选hash，后端必须hash）
- CSRF防护
- 验证码有效期：5分钟
- 登录失败次数限制：5次/10分钟
- 滑动验证码（防机器人）

## 性能优化
- 3D场景延迟加载（优先渲染表单）
- 输入验证防抖：500ms
- 唯一性检查防抖：800ms
- 图片压缩

## 错误处理
- 网络错误：Toast提示 + 重试按钮
- 验证失败：字段下方显示错误信息
- 第三方登录失败：Toast提示 + 回退到登录页

## 埋点需求
- 登录/注册转化率
- 第三方登录使用率（微信 vs QQ）
- 验证码发送成功率
- 表单验证失败类型统计
- 密码重置使用率
- 3D动画加载成功率（设备兼容性）
