# M1 首页规范

## 页面概述
- **路由**: `/` 或 `/home`
- **优先级**: Phase 2（核心搜索与方案生成）
- **依赖**: 登录/注册系统
- **设计参考**: App Store 风格

## 功能模块

### 1. Header（头部导航）
- **Logo区域**
  - 显示"盘根 AI 指南针"品牌标识
  - 点击返回首页
  - 集成领域切换下拉菜单（影视创作、编程开发、通用办公）
- **用户信息**
  - 未登录：显示"登录/注册"按钮
  - 已登录：显示用户头像 + 下拉菜单
    - 个人中心
    - 后台管理（仅编辑/管理员可见）
    - 护眼模式开关
    - 退出登录
- **语言切换**
  - 支持中、日、英、俄四种语言
  - 记忆用户偏好（localStorage + 云端同步）

### 2. AI搜索框
- **功能**
  - 搜索AI工具或输入问题
  - 占位符文本：根据语言动态显示
  - 搜索历史下拉提示（最近5条）
  - 智能联想（根据输入实时显示建议）
- **交互**
  - 点击：展开搜索历史
  - 输入：触发AI智能搜索
  - 回车：跳转到搜索结果页（M2）
- **API需求**
  - `GET /api/search/history` - 获取用户搜索历史
  - `POST /api/search/suggest` - 获取智能联想
  - `POST /api/search` - 执行搜索

### 3. 模块切换器
- **选项**
  - 工具推荐（默认）
  - 经验心得
- **交互**
  - 点击切换：200ms fade动画
  - 视觉反馈：选中项高亮 + 阴影
- **状态管理**
  - `contentType: 'tool' | 'experience'`

### 4. 我的工具（折叠卡片）
- **显示内容**
  - 已选工具数量：如"我的工具 (3)"
  - 默认折叠状态
  - 展开后显示工具列表（横向滚动）
- **交互**
  - 点击展开/收起：滑动动画 + Light Haptic
  - 点击工具卡片：跳转到工具详情页（M2）
- **API需求**
  - `GET /api/user/tools` - 获取用户已选工具
- **状态管理**
  - `selectedToolIds: Set<string>`

### 5. 工具推荐区域
- **精选方案（横向滚动）**
  - App Store 风格卡片
  - 每张卡片包含：
    - 工具图标/封面
    - 工具名称
    - 简短描述
    - 评分 + 使用人数
  - 点击卡片：跳转到工具详情页
- **热门工具（垂直列表）**
  - 标题："热门工具"
  - 网格布局：移动端2列，平板3列，桌面4列
  - 每个工具卡片：
    - 工具封面图
    - 工具名称
    - 简短描述
    - 评分 + 标签
    - 选择按钮（勾选框）
- **API需求**
  - `GET /api/tools/featured?domain={domain}` - 获取精选工具
  - `GET /api/tools?domain={domain}` - 获取工具列表

### 6. 经验心得区域
- **子选项卡**
  - 精品专区（Featured）
  - 大众广场（Plaza）
- **精品专区内容**
  - 系列主题（Topics）
    - 网格布局（3列）
    - 主题卡片：封面 + 标题 + 文章数量 + 评分
    - 点击：跳转到主题首篇文章
  - 精选文章（独立文章）
    - 列表布局
    - 文章卡片：封面 + 标题 + 摘要 + 作者 + 阅读时间
    - 点击：跳转到阅读页（M3）
- **大众广场内容**
  - 用户投稿文章（未精选）
  - 列表布局
  - 空状态："暂无内容，快来投稿吧！"
- **API需求**
  - `GET /api/topics?domain={domain}` - 获取主题列表
  - `GET /api/articles?domain={domain}&featured=true` - 获取精选文章
  - `GET /api/articles?domain={domain}&featured=false` - 获取广场文章

### 7. 浮动工具栏（Floating Dock）
- **显示条件**
  - 仅当 `selectedToolIds.size > 0` 时显示
- **位置**
  - 固定在页面底部中央
  - 毛玻璃效果 + 阴影
- **内容**
  - 左侧：已选工具预览（最多显示4个工具图标）
  - 中间：工具数量显示
  - 右侧：操作按钮
    - 清空选择（垃圾桶图标）
    - 分享（分享图标）
      - 复制文本
      - 分享到微信
      - 分享到QQ
    - 生成方案（主按钮，渐变背景）
- **交互**
  - 出现：从底部滑入 + 渐显（300ms）
  - 点击生成方案：跳转到M5生成方案页
  - 点击分享：弹出分享菜单
- **状态管理**
  - `dockCopied: boolean` - 复制成功提示
  - `showDockShareMenu: boolean` - 分享菜单显示状态

### 8. Footer（页脚）
- **内容**
  - ICP备案号：蜀ICP备2025165644号-1
  - 公安备案：备案中
  - 版权信息
  - 友情链接（可选）
- **样式**
  - 灰色小字（12px）
  - 固定在页面最底部

## 状态管理

```typescript
interface HomePageState {
  // 视图状态
  view: ViewState;
  themeMode: ThemeMode; // 'light' | 'eye-care'
  language: Language; // 'zh' | 'en' | 'jp' | 'ru'
  
  // 领域与内容
  currentDomain: Domain; // 'creative' | 'dev' | 'work'
  contentType: ContentType; // 'tool' | 'experience'
  experienceTab: ExperienceTab; // 'featured' | 'plaza'
  
  // 用户状态
  isLoggedIn: boolean;
  
  // 工具选择
  selectedToolIds: Set<string>;
  
  // 浮动工具栏
  dockCopied: boolean;
  showDockShareMenu: boolean;
}
```

## API接口

### 获取工具列表
```
GET /api/tools?domain={domain}&featured={boolean}
Response: Tool[]
```

### 获取主题列表
```
GET /api/topics?domain={domain}
Response: Topic[]
```

### 获取文章列表
```
GET /api/articles?domain={domain}&featured={boolean}&topicId={id}
Response: Article[]
```

### 获取用户已选工具
```
GET /api/user/tools
Response: { toolIds: string[] }
```

### 搜索历史
```
GET /api/search/history
Response: { queries: string[], timestamp: string }[]
```

## 响应式设计

### 移动端 (<768px)
- 单列布局
- 工具网格：2列
- 浮动工具栏宽度：95vw
- Header：收缩Logo，隐藏语言切换（移到设置）

### 平板 (768px-1024px)
- 工具网格：3列
- 侧边栏可收缩

### 桌面 (>1024px)
- 最大宽度：1280px（内容区）
- 工具网格：4列
- 左侧导航栏固定显示

## 动画规范
- 页面加载：fade-in 500ms
- 模块切换：fade 200ms
- 卡片hover：scale(1.02) + shadow 150ms
- 浮动工具栏出现：slide-in-from-bottom + fade 300ms

## 交互反馈
- 按钮点击：Light Haptic（移动端）
- 工具选择：Medium Haptic
- 错误提示：Error Haptic + Toast

## 性能优化
- 图片懒加载（Intersection Observer）
- 工具列表虚拟滚动（当数量>50时）
- 搜索防抖：300ms
- 预加载：鼠标hover卡片时预加载详情数据

## 错误处理
- 网络错误：显示重试按钮
- 空状态：友好提示文案 + 插图
- 加载失败：骨架屏 → 错误提示

## 埋点需求
- 页面访问量（PV/UV）
- 工具卡片点击率
- 搜索使用率
- 浮动工具栏转化率
- 领域切换统计
