## 1. 产品定位

> 盘根·AI 指南针是一个熵减器——把 AI 时代的混乱信息，通过实践验证，转化为有序的、可用的知识。

**三句话核心**：

1. **一半生活，一半理想** — 校园站（生活面）服务日常，全球站（理想面）服务创新
2. **从实验到应用** — idea 在理想站被探索，经过实践验证后，沉淀为生活站里普通人能直接用的知识
3. **推动 AI 大众化** — 帮助人们正确认识和使用 AI，实现个人价值与社会价值的统一

**底层驱动**：Ferry 框架（熵减思维：规划→执行→复盘）指导内容方向与质量保证。

## 2. 双站架构

双前端 + 单后端，两站共享后端逻辑，通过站点配置区分行为。部署时可拆为两个实例，代码不变。

| 属性    | xyzidea.cn（校园站·生活面） | xyzidea.com（全球站·理想面） |
| ----- | ------------------- | -------------------- |
| 当前用户  | 本校学生和老师             | 线上创作者、程序员等创作类工作者     |
| 内容领域  | 日常起居 / 成长提升 / 精明消费  | 影视创作 / 编程开发 / 通用办公   |
| 登录方式  | 手机号 + 验证码           | 申请制 + 邀请制 + GitHub OAuth |
| 服务器   | 中国大陆（ICP 备案）        | 香港/新加坡               |
| 待验证假设 | 扩展到校园所在区域的所有人       | 扩展到全球想用 AI 解决问题的人    |

**数据隔离红线**：cn 用户数据绝不流向海外；com 站禁止收集中国敏感个人信息；两站账号体系完全独立。

## 3. 用户画像与知识流

> 基于实际调查数据，非假设。

| 画像         | 来源   | 痛点                                             | 需求                      |
| ---------- | ---- | ---------------------------------------------- | ----------------------- |
| 本校学生       | 校园调查 | 不知道怎么用 AI 解决日常问题                               | 食堂、选课、省钱等场景的 AI 实用教程    |
| 本校老师       | 校园调查 | 想用 AI 辅助教学但不知从何入手                           | 备课、出题、科研等场景的 AI 实用教程   |
| 线上创作者      | 线上调研 | 有 idea 但缺好用工具及落地方法 | AI 音乐、AI 绘画、AI 视频的实验与案例 |
| 程序员/创作类工作者 | 线上调研 | 工具多但工作流不稳定                                     | 以目标为中心的工具链，可复用方案        |

**知识流**：用户不按站划分。同一个人的 idea 在理想站被探索，实践验证有用后，沉淀到生活站给普通人直接用。形成 **实验 → 验证 → 应用** 的转化闭环。

## 4. 功能清单

> 核心功能对称，差异在登录与内容管理。

### 共享功能（两站一致）

| 模块   | 功能说明                            |
| ---- | ------------------------------- |
| 首页   | 品牌展示 + 搜索入口 + 领域切换 + 内容分类浏览     |
| 搜索   | 普通搜索（免费）+ AI 搜索（消耗次数）           |
| 内容浏览 | 专题和文章的卡片式浏览                     |
| 资讯     | com 站：全球 AI 前沿资讯；cn 站：当地实时消息  |
| 文章阅读 | 正文阅读，系列文章支持目录导航                 |
| 方案   | 生成方案、保存、导出（md/txt/csv）          |
| 收藏   | 收藏工具/文章，收藏管理                    |
| 用户后台 | 个人资料、我的方案、收藏夹、我的额度、设置           |
| 站间链接 | 跳转至对方站点                         |

### 国内站特有

| 模块   | 功能说明           |
| ---- | -------------- |
| 登录   | 手机号注册/登录       |
| 学生认证 | 学生认证申请、审核、等级成长 |
| 校园专区 | 学生认证后解锁的校园专属内容 |

### 全球站特有

| 模块    | 功能说明                     |
| ----- | ------------------------ |
| 登录    | 申请制 + 邀请制 + GitHub OAuth |
| 内容工作室 | 内容编辑与管理                  |
| 工具展览 | 精选好工具的沉浸式展示，引导用户探索 |

### 校园内容领域（cn 站）

| 领域   | 覆盖内容           |
| ---- | -------------- |
| 日常起居 | 食堂测评、出行攻略、宿舍指南 |
| 成长提升 | 选课攻略、社团招新、社交活动 |
| 精明消费 | 教材省钱、二手交易、避坑指南 |

### 后台（管理端）

| 模块     | 功能说明                       |
| ------ | -------------------------- |
| 内容管理   | 文章、专题、工具的增删改查与上下架          |
| 用户管理   | 用户列表、角色分配、账号状态             |
| 申请审核   | com 站注册申请审批、学生认证审核         |
| 数据看板   | 用户量、浏览量、方案生成量、AI 调用量等核心指标  |
| 内容分析   | 指定内容集的 AI 分析，生成质量报告与统计（可导出供上报） |
| 系统设置   | 站点配置、权限管理、操作日志             |

### 中台（共享服务层）

| 模块   | 功能说明                             |
| ---- | -------------------------------- |
| AI 网关 | 代理智谱 API 调用，统一计费与限流              |
| 智能分析 | 内容质量分析 + 用户需求分析 + 创作方向建议，驱动内容策略 |
| 内容同步 | cn → com 单向同步（文章、专题、工具），原型阶段脚本同步 |
| 搜索服务 | 关键词搜索 + AI 语义搜索的统一入口             |
| 通知服务 | 邮件通知（com 站审核结果、邀请链接）与站内消息       |

## 7. 商业原则

- 收费以覆盖模型调用与运营成本为底线，不以高利润为导向
- 当前以个人内容输出为主，未来向高质量平台演进
- 自媒体引流：通过抖音/小红书等平台发布 AI 用法内容，引流至网站查看完整教程与方案

---

**项目归类**：科教互联网 + 自媒体 + 区域化管理
**备案**：蜀ICP备2025165644号-1 · 公安备案中
**合规**：PIPL / 数据安全法 / 网络安全法 / GDPR
