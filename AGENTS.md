## 快速参考

- **代码目录**：`NorthStar/packages/`（shared / frontlife-web / frontai-web / server）
- **文档目录**：`specs/`（使命 + PRD + 实现规格）
- **文档层级**：宪法 → 使命（MISSION.md）→ PRD → 实现规格
- **安装**：`cd NorthStar && pnpm install`
- **校园站**：`cd NorthStar/packages/frontlife-web && pnpm dev` → localhost:3001
- **全球站**：`cd NorthStar/packages/frontai-web && pnpm dev` → localhost:3000
