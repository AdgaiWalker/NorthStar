# 设置模块（Settings）原型设计分析

## 1. 整体布局设计

设置界面采用经典的**左右分栏布局**，结构清晰，易于扩展。

- **容器尺寸**：弹窗式设计（Modal），带有圆角和柔和的阴影，右上角包含关闭按钮 `×`。
- **左侧导航栏 (Sidebar)**：
  - **宽度**：固定宽度（约占整体宽度的 25%）。
  - **菜单项**：垂直排列，包含图标 + 文字标签。
    - *Appearance*（外观）
    - *AI Engine*（AI 引擎）
    - *Persona*（角色/人设）
    - *Guide*（指南）
  - **选中态**：高亮背景色（浅蓝）+ 蓝色文字 + 蓝色图标。
  - **底部操作**：*Reset to Default*（重置默认）按钮固定在侧边栏底部。
- **右侧内容区 (Content Area)**：
  - 占据剩余空间，白色背景。
  - 内容按组块垂直排列，组块间有明显的留白。

## 2. 模块功能详解

### 2.1 外观设置 (Appearance)

此模块用于个性化界面展示，包含四个核心配置项：

**A. Language (语言)**
- **控件类型**：分段控制器（Segmented Control）/ 胶囊切换器。
- **选项**：English（选中态为白底蓝框蓝字）、中文。
- **交互**：点击直接切换，互斥选择。

**B. Theme (主题)**
- **控件类型**：卡片式单选组（Card Selection）。
- **选项**：
  - *Light*（亮色）：选中态，带蓝色边框和高亮图标。
  - *Dark*（暗色）：未选中态，灰色图标。
  - *Tea*（护眼/茶色）：未选中态。
- **视觉反馈**：图标 + 文字说明，直观展示主题风格。

**C. Font (字体)**
- **Font Style**：标准下拉菜单（Dropdown），当前显示 "System Font (Sans-serif)"。
- **Article Font Size**：滑动条（Slider）。
  - 左侧为进度条（蓝色），右侧为轨道（灰色）。
  - 滑块为圆形，支持拖拽。
  - 右侧实时显示数值 "15px"。

**D. Editor (编辑器)**
- **Default File Format**：下拉菜单，用于选择默认导出/保存格式（如 Markdown .md）。
- **Smart Format Conversion**：
  - **控件**：开关（Toggle Switch）。
  - **状态**：开启状态显示为蓝色背景，滑块在右。
  - **辅助文本**：下方有一行灰色小字说明功能细节（"When enabled, MD to TXT will remove Markdown markup..."），对用户非常友好，减少困惑。

### 2.2 AI 引擎设置 (AI Engine)

此模块专注于本地大模型的管理与配置。

**A. AI Model Management (模型管理)**
- **提示信息栏 (Info Alert)**：
  - 浅灰色背景，左侧有蓝色竖条强调。
  - 文本内容说明内置模型情况及手动添加支持。
- **Model Storage Location (模型存储位置)**：
  - 显示标签。
  - 操作按钮：*Open Models Folder*（打开模型文件夹），采用浅蓝色背景 + 蓝色图标/文字，视觉权重次于主按钮但高于纯文本链接。

**B. Installed Models (已安装模型)**
- **展示形式**：列表卡片。
- **卡片内容**：
  - **模型名称**：qwen2.5:0.5b。
  - **标签 (Tags)**：
    - *Built-in*（内置）：浅绿色背景。
    - *Current*（当前使用）：亮蓝色背景。
  - **元数据**：显示大小（397 MB）和安装时间（4 hours ago）。

**C. Available Models (可用模型)**
- **展示形式**：详细列表项。
- **卡片内容**：
  - **头部**：名称 + Built-in 标签 + 大小（灰色胶囊背景）。
  - **描述**：简短的功能描述（"Lightning fast, lightweight..."）。
  - **状态**：左下角显示带对勾的 *Installed*（已安装）状态，绿色高亮，表明无需再次下载。

## 3. 视觉与交互规范总结

1.  **色彩体系**：
    - **主色 (Primary)**：蓝色（用于选中态、开关开启、主要按钮、进度条）。
    - **功能色**：绿色（用于状态标识，如 Installed, Built-in）。
    - **文本色**：深灰（标题）、中灰（正文）、浅灰（辅助说明）。
    - **背景色**：纯白（内容区）、浅灰（侧边栏/背景）。
2.  **圆角 (Radius)**：
    - 整体界面使用较大的圆角（约 12px-16px），风格现代、柔和。
    - 按钮和输入框使用中等圆角（约 6px-8px）。
3.  **排版 (Typography)**：
    - 标题加粗，层级分明。
    - 辅助文字字号较小且颜色变浅，避免喧宾夺主。
4.  **交互反馈**：
    - 选中状态通过“边框颜色 + 背景色 + 图标颜色”三重变化来强化。
    - 开关和滑块提供清晰的视觉状态。

此设计展现了一个成熟、现代的桌面应用设置中心，兼顾了易用性与专业性（特别是针对本地 AI 模型的管理）。
