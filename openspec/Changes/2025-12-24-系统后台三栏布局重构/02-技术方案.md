# 技术方案：系统后台三栏布局

## 一、整体布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│                        100vw × 100vh                                │
├────────┬──────────────────┬─────────────────────────────────────────┤
│ 1栏    │      2栏         │              3栏                        │
│ 56px   │    240px         │           flex: 1                       │
│ fixed  │   可收起→0       │          主内容区                       │
├────────┼──────────────────┼─────────────────────────────────────────┤
│        │                  │                                         │
│ 图标   │  上下文导航       │  表格 / 表单 / 编辑器 / 图表            │
│ 导航   │  或筛选器        │                                         │
│        │                  │                                         │
│        │                  │                                         │
│────────│                  │                                         │
│ 👤     │  返回前台        │                                         │
│ 底部   │                  │                                         │
└────────┴──────────────────┴─────────────────────────────────────────┘
```

## 二、1栏（Super Sidebar）规格

### 2.1 尺寸
- 宽度：56px（固定）
- 高度：100vh
- 背景：`bg-slate-900` 或 `bg-zinc-900`（深色）

### 2.2 图标分组
```
┌──────────┐
│ 🏠 Logo  │  ← 点击返回前台首页
├──────────┤
│ 内容区    │

│ 📋 管理   │
│ ✅ 审核   │
│ 🛠 工具   │
├──────────┤  ← 分隔线
│ 用户区    │

├──────────┤  ← 分隔线
│ 数据区    │

├──────────┤  ← 分隔线
│ 运营区    │

├──────────┤

└──────────┘
```

### 2.3 图标交互
- **默认态**：图标 24px，颜色 `text-slate-400`
- **悬停态**：背景 `bg-slate-800`，图标 `text-white`
- **激活态**：左侧 3px 蓝色竖条，背景 `bg-slate-800`，图标 `text-white`
- **Tooltip**：悬停 300ms 后显示模块名称

### 2.4 图标与路由映射

## 三、2栏（Context Sidebar）规格

### 3.1 尺寸
- 宽度：240px（默认），可收起至 0px
- 高度：100vh
- 背景：`bg-slate-50` 或 `bg-zinc-50`（浅色）

### 3.2 收起/展开
- 收起按钮：位于 2栏右上角，图标 `ChevronLeft`
- 展开按钮：当 2栏收起时，在 1栏右侧边缘显示 `ChevronRight`
- 动画：`transition-all duration-200 ease-out`

## 四、3栏（Main Content）规格

### 4.1 尺寸
- 宽度：`flex: 1`（剩余空间）
- 最小宽度：600px
- 高度：100vh
- 背景：`bg-white`

### 4.2 内容工作室的特殊布局（3栏再分三子栏）

## 六、关键组件

### 6.1 AdminLayout
- 职责：渲染三栏布局骨架，管理 2栏收起状态
- 状态：`isSidebarCollapsed: boolean`
- 子组件：`SuperSidebar`, `ContextSidebar`, `Outlet`

### 6.2 SuperSidebar
- 职责：渲染 1栏图标导航
- Props：`activeModule: string`, `onModuleChange: (module) => void`

### 6.3 ContextSidebar
- 职责：根据当前模块渲染对应的 2栏内容
- Props：`module: string`, `isCollapsed: boolean`, `onCollapse: () => void`

### 6.4 权限控制
- 根据用户角色隐藏无权限的图标
- `superadmin`：所有模块
- `admin`：除角色管理外的所有模块
- `editor`：内容工作室、内容管理
- `reviewer`：审核工作台

## 八、交互规格

### 8.1 拖拽排序（仅内容工作室）
- **同级排序**：拖拽文件/文件夹，放下后更新 `sortIndex`
- **跨文件夹移动**：拖拽到文件夹上悬停 500ms，文件夹展开（Spring-loading）
- **视觉反馈**：
  - 拖拽中：源项目半透明，鼠标显示拖拽图标
  - 放置目标：蓝色边框高亮
  - 插入位置：蓝色横线指示器

### 8.2 键盘快捷键
- `Esc`：从沉浸模式返回（如收起的侧边栏展开）
- `Ctrl/Cmd + K`：全局命令面板（后续迭代）
- `Ctrl/Cmd + S`：保存当前编辑

### 8.3 返回前台
- 点击 1栏顶部 Logo
- 点击 2栏底部"返回前台"按钮
- 路由跳转至 `/`
