```typescript

export const useShare = () => {
  const [copied, setCopied] = useState(false);
  
  const copyToClipboard = async (text: string) => {
    await navigator.clipboard.writeText(text);
    setCopied(true);
    setTimeout(() => setCopied(false), UI_CONSTANTS.DELAYS.COPY_FEEDBACK);
  };
```

### 10. 命名不一致问题

**问题描述:** 同一概念使用了多种命名方式

**位置及示例:**

**10.1 用户相关命名不一致**

**10.3 主题相关命名**
- `themeMode` (状态)
- `isEyeCare` (派生值)
- 建议统一命名模式

**违反原则:** 命名一致性

**严重程度:** 🟡 中

**建议修复:**
- [ ] 制定命名规范文档
- [ ] 统一 boolean 前缀 (is/has/should)
- [ ] 统一事件处理器前缀 (on/handle)
